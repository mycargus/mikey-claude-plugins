# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the mikey-claude-plugins project. ADRs document significant decisions along with their context and consequences.

## Creating a New ADR

1. Copy `adr-template.md` to `adr-NNN-short-title.md` (zero-padded, next sequential number).
2. Fill in all sections. Keep it concise.
3. Add an entry to the index table above.
